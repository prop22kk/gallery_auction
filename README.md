# 🎨 Gallery Auction

### 온라인 예술 작품 경매 시스템

---

## 📌 프로젝트 개요

**Gallery Auction**은 사용자가 온라인으로 예술 작품을 경매에 등록하고, 입찰 및 낙찰까지의 과정을 웹을 통해 진행할 수 있는 예술 경매 시스템입니다.

---

## 👥 프로젝트 구성원 및 역할

- **김지현**: 회원(postman)연동, 프론트엔드&백엔드 연결

- **이지영**: 

---

## 🛠️ 사용 기술 스택

- **백엔드**: Java, Spring Boot, Spring Web, Spring JDBC  
- **프론트엔드**: Svelte (VSCode 환경에서 개발)  
- **데이터베이스**: PostgreSQL  
- **API 테스트 및 연동**: Postman  
- **개발 환경**: IntelliJ IDEA, Visual Studio Code

---

## 🔑 주요 기능

- **회원 관리**
  - 신규 회원 등록
  - 회원 정보 조회 및 수정
  - 프론트엔드와 연동 완료 (Svelte 연결 진행 중)
